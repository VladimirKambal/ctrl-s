# Ctrl + S
Это мой первый микро-проект написанный на Python, здесь моей главной задачей было упростить свою жизнь в написании кода.

Кртакая предыстория. Мне было очень лень открывать консоль каждый раз после изменений в проекте, лень прописывать дирректорию, git commit и так далее.
Так же у меня была плохая привычка — вместо того чтобы работать с Git-ом, я нажимал сочетание клавиш ctrl+s (от сюда и название).
Таким образом, когда дело доходило до push-а, колличество измененных строк могло перевалить за 100, что абсолютно не хорошо.

Этот микро-проект должен упростить мою жизнь. Теперь, при запущенном коде и каждом нажатии ctrl+s, высвечивается командная строка, просящая ввести название коммита, и автоматически push-а его.
Благодаря этому, скачав main.py, изменив всего 1 строку кода, и запустив его в консоли от имени администротора, мне не приходится лишний раз заботиться о колличестве измененных строк.

## Установка
Перед началом не забудьте установить зависимости

```
py -m pip keyboard
py -m pip tempfile
```

Скачиваете последнюю версию zip-архива с файлами, разархивируйте содержимое в удобную вам дирректорию.

## Настройка
Найдите строчку
```python
REPO_PATH = r"C:\Users\thinzano\Desktop\git project\project 3"
```
И замените внутренность кавычек на полный путь к вашему проекту. Сохраните изменения.

## Запуск
После этого можно запускать командную строку, обязательно(!) от имени администратора, в противном случае библиотека "keyboard" не будет улавливать нажатие клавиш когда открыто другое окно.

```
cd (папка с файлами)
py main.py
```

Все готово! Теперь при нажатии Ctrl+s должна появиться консоль, которая попросит ввести название коммита, а после нажатия enter автоматически запушит его.
